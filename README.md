# IT Service And Technology site

A small static website for IT services and technology information.

## Project structure

- `index.html` — Main HTML page
- `javascript.js` — Site JavaScript
- `style.css` — Main styles
- `media.css` — Responsive / media-specific styles
- `images/` — Image assets used by the site

## Quick start

1. Open `index.html` in your browser to view the site locally.

2. (Optional) Serve with a simple HTTP server for local testing:

- Python 3:

```
python -m http.server 8000
```

- Node (http-server):

```
npm install -g http-server
http-server -p 8000
```

Then open http://localhost:8000 in your browser.

## Development

- Edit `index.html`, `javascript.js`, `style.css`, and `media.css` directly.
- Keep assets in the `images/` folder.
- Use your browser devtools to inspect and debug styles and scripts.
